# SGVED

Sistema gestor de votaciones y encuestas docentes.

## Tecnologías
SGVED consta de las siguientes tecnologías:
- Springboot 2.5.0-SNAPSHOT
- Bootstrap 5.0.0-beta2
- jQuery 3.6.0
- H2 Database 
- Thymeleaf
